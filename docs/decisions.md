# Berkeley Swim Decisions Log

Use this file to capture project decisions as Berkeley Swim grows. These notes should make the README, resume bullet, and later project writeup easier to write.

## North Star

Mission: Berkeley Swim helps Berkeley residents quickly see whether they can swim now, what's open next, and where to buy the right pass without digging through dense city PDFs or confusing catalog pages.

Homepage tagline: An unofficial, faster way to see what's open at Berkeley's public pools.

## Decisions

### 2026-04-30 - Phase 0 pre-flight

- Project name: Berkeley Swim.
- Launch plan: use the free Vercel subdomain first, likely `berkeley-swim.vercel.app`; revisit a custom domain only after the app has real users.
- Product stance: civic, unofficial, and useful; do not use City of Berkeley branding, handle payments, proxy accounts, or imply city affiliation.
- MVP priority: answer the two questions swimmers actually have: "Can I swim now?" and "How do I buy the right pass?"
- Scope guardrail: ship the schedule and pass-helper experience before adding stretch features like notifications, crowd reports, or multi-pool expansion.
